# Tetris

Neon cyberpunk Tetris game built with HTML5 Canvas.

## Features

- Classic Tetris gameplay
- Neon cyberpunk visual style with glow effects
- Next piece preview
- Score, level, and lines tracking
- Ghost piece preview
- Pause functionality

## Controls

- **← →** - Move piece
- **↑** - Rotate piece
- **↓** - Soft drop
- **Space** - Hard drop
- **P** - Pause
- **Enter** - Restart (when game over)

## Run Locally

```bash
npx serve .
```

Or open `index.html` directly in a browser.

## Tech Stack

- Single HTML file
- Vanilla JavaScript
- HTML5 Canvas
- No external dependencies
